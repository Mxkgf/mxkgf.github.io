选择名称

名称的可互换性并不意味着正式的参数名称根本不重要。相反，精心选择的函数和参数名称对于函数定义的人类可解释性至关重要！

以下指南改编自 Python 代码的风格指南，该指南可作为所有（非叛逆的）Python 程序员的指南。一组共享的约定使开发人员社区成员之间的通信顺畅。作为遵循这些约定的副作用，您会发现您的代码变得更加内部一致。

  1. 函数名称为小写，单词之间用下划线分隔。鼓励使用描述性名称。
  2. 函数名通常唤起由解释器应用于参数的操作（例如，print，add，square）或结果数量的名称（例如，max，abs，sum）。
  3. 参数名称为小写，单词之间用下划线分隔。最好使用单字名称。
  4. 参数名称应唤起参数在函数中的作用，而不仅仅是允许的参数类型。
  5. 当单字母参数名称的作用很明显时，它们是可以接受的，但要避免使用“l”（小写 L）、“O”（大写字母 o）或“I”（大写字母 i），以避免与数字混淆。

这些准则有许多例外，即使在 Python 标准库中也是如此。就像英语的词汇一样，Python 继承了各种贡献者的单词，结果并不总是一致的。

作为抽象的函数

虽然它非常简单，但sum_squares体现了用户定义函数最强大的属性。函数sum_squares是根据函数square定义的，但只依赖于square在其输入参数和输出值之间定义的关系。

我们可以写作 sum_squares 而不用担心如何平方一个数字。可以抑制如何计算平方的细节，稍后再考虑。事实上，就 sum_squares 其而言， square 它不是一个特定的函数体，而是函数的抽象，即所谓的函数抽象。在这个抽象级别上，任何计算平方的函数都同样好。

因此，仅考虑它们返回的值，以下两个用于对数字进行平方的函数应该是无法区分的。每个都接受一个数字参数，并生成该数字的平方作为值。

换句话说，函数定义应该能够抑制细节。该函数的用户可能不是自己编写的函数，而是作为“黑匣子”从另一个程序员那里获得的。程序员不需要知道函数是如何实现的就可以使用它。Python 库具有此属性。许多开发人员使用那里定义的函数，但很少有人检查它们的实现。

函数抽象的各个方面。为了掌握函数抽象的使用，考虑它的三个核心属性通常是有用的。函数的域是它可以采用的参数集。函数的范围是它可以返回的值集。函数的意图是它计算输入和输出之间的关系（以及它可能产生的任何副作用）。通过函数抽象的域、范围和意图来理解函数抽象对于在复杂程序中正确使用它们至关重要。

编程要素

当我们描述一种语言时，我们应该特别注意语言提供的将简单思想组合成更复杂思想的方法。每种强大的语言都有三种这样的机制：

  • 原始表达式和语句，它们表示语言提供的最简单的构建块，
  • 组合方式，通过这种组合方式，组合元素由更简单的元素构建而成，以及
  • 抽象手段，通过这种手段可以将复合元素作为单元进行命名和操作。

在编程中，我们处理两种元素：函数和数据。（很快我们就会发现它们真的没有那么明显。非正式地，数据是我们想要操作的东西，函数描述了操作数据的规则。）因此，任何强大的编程语言都应该能够描述原始数据和原始函数，并具有一些组合和抽象函数和数据的方法。

调试的指导原则

  1. 增量测试：每个编写良好的程序都由可以单独测试的小型模块化组件组成。尽快尝试您编写的所有内容，以便及早发现问题并获得对组件的信心。

  2. 隔离错误：语句输出中的错误通常可归因于特定的模块化组件。尝试诊断问题时，请先将错误跟踪到最小的代码片段，然后再尝试更正错误。

  3. 检查你的假设：解释器确实不折不扣地执行你的指示——不多也不少。当某些代码的行为与你认为的行为不匹配时，检查您的假设，然后将调试工作重点放在验证您的假设是否确实成立上。

  4. 咨询他人：如果您不理解错误消息，请咨询朋友、教师或搜索引擎。如果您已经隔离了一个错误，但不知道如何纠正它，请让其他人查看。

    希望增量测试、模块化设计、精确的假设和团队合作能贯穿您的整个计算机科学生涯。

Amdahl定理

当我们对系统的某个部分加速时，其对系统整体性能的影响取决于该部分的重要性和加速程度。要想显著加速整个系统，必须提升全系统中相当大的部分的速度。

并发和并行

  1. 线程级并发
  2. 指令级并行
  3. 单指令、多数据并行

概念

  1. 并发 (concurrency)指一个同时具有多个活动的系统;
  2. 并行 (parallelism)指的是用并发来使一个系统运行得更快

计算机系统中抽象的重要性

  1. 文件是对I/0设备的抽象
  2. 虚拟内存是对程序存储器的抽象
  3. 进程是对一个正在运行的程序的抽象
  4. 虚拟机，它提供对整个计算机的抽象，包括操作系统、处理器和程序

极限导论

概要

  • 对于极限是什么的直观概念
  • 左、右与双侧极限、及无穷和负无穷处的极限
  • 何时极限不存在
  • 三明治定理（也称作“夹逼定理”）

极限：基本思想

$\lim\limits_{x\to2}f(x)=1$读作“当 $x$ 趋于 2, $f(x)$ 的极限等于 1,还可以写作$f(x)\to1\text{当}x\to2$

左极限与右极限

仅当左极限和右极限在 x = a 处都存在且相等存在双侧极限，否则双侧极限不存在,写作DNE

垂直渐近线

$f在x=a处有一条垂直渐近线说的是，\lim\limits_{x\to a^+}f(x)和\lim\limits_{x\to a^-}f(x)处至少有一个极限是\infty或-\infty$

在无穷和负无穷处的极限

$f在y=L处有一条右侧水平渐近线意味着\lim\limits_{x\to\infty}f(x)=L$

$f在y=M处有一条左侧水平渐近线意味着\lim\limits_{x\to-\infty}f(x)=M$

大的数和小的数

  • 如果一个数的绝对值是非常大的数, 则这个数是大的
  • 如果一个数非常接近于 0(但不是真的等于 0), 则这个数是小的

关于渐近线的常见误解

  1. 一个函数不一定要在左右两边有相同的水平渐近线

    $y = \arctan (x)在 y = π/2 处有一条右侧水平渐近线, 在 y = -π/2 处有一条左侧水平渐近线$

  2. 一个函数不可能和它的渐近线相交

    $f(x)=\large\frac{\sin(x)}{x}\normalsize的函数f,\lim\limits_{x\to\infty}\large\frac{\sin(x)}{x}=\normalsize0$
    $\large\frac{\sin(x)}{x}\normalsize和\sin(x)有相同的零点, 即 π, 2π, 3π, ···$

三明治定理

$如果对于所有在a附近的x都有g(x)\le f(x)\le h(x),且\lim\limits_{x\to a}g(x)=\lim\limits_{x\to a}h(x)=L,则\lim\limits_{x\to a}f(x)=L$

函数

函数是将一个对象转化为另一个对象的规则。起始对象称为输入，来自称为定义域的集合，返回对象称为输出，来自称为上域的集合。

一个函数必须给每一个有效的输入指定唯一的输出。

区间表示法

区间表示法 集合描述法
$(a, b)$ {$x:a<x<b$}
$(a,b)$ {$x:a<x<b$}
$[a,b]$ {$x:a\le x\le b$}
$(a,b]$ {$x:a< x\le b$}
$[a,b)$ {$x:a\le x<b$}
$(a,+\infty)$ {$x:x>a$}
$[a,+\infty)$ {$x:x\ge a$}
$(-\infty,b)$ {$x:x<b$}
$(-\infty,b]$ {$x:x\le b$}
$(-\infty,+\infty)$ $\mathbb{R}$

求定义域

  1. 分数的分母不能为零
  2. 不能取负数的平方根
  3. 不能取负数或零的对数

利用图像求值域

根据定义域画出函数图像，然后想象从图像的左边和右边很远的地方朝向 y 轴水平地射入两束亮光. 曲线会在 y 轴上有两个影子, 一个在 y 轴的左侧, 另一个在 y 轴的右侧. 值域就是影子的并集。

垂线检验

如果没有一条垂线和图像相交多于一次, 那么是函数的图像。

反函数

反函数的定义

$\text{设函数}y=f(x)\text{的定义域是}D,\text{值域是}f(D)。\text{则}y=f(x)\text{的反函数为:}x=f\mathop{}\nolimits^{-1}(y),y\in f(D)$

反函数的性质

  • $f\mathop{}\nolimits^{-1}(f(x))=x$
  • $f(f\mathop{}\nolimits^{-1}(x))=x$
  • $f(f\mathop{}\nolimits^{-1}(f(x)))=y$

水平线检验

如果每一条水平线和一个函数的图像相交至多一次, 那么这个函数就有一个反函数

函数的复合

$\text{有}f(x) = cos(x^2),\text{令}g(x)=x^2,h(x)=cos(x),\text{故我们可以写出}f(x)=h(g(x)),\text{也可以表示为}f=h\circ g,$$\text{即}f\text{是}g\text{与}h\text{的复合}$

奇函数和偶函数

$\text{一般而言, 一个函数可能是奇的, 可能是偶的, 也可能非奇非偶. 要记住这一点, 大多数函数是非奇非偶的.}$
$\text{另一方面, 只有一个函数是既奇又偶的, 它就是非常单调的对所有}x 都成立的 f (x) = 0(我们称之为零函数)$

  • 偶函数的图像关于 y 轴具有镜面对称性
  • 奇函数的图像关于原点有 180° 的点对称性
  • $\text{对于偶函数}f(x)=f(-x)$
  • $\text{对于奇函数}f(x)=-f(-x)$
